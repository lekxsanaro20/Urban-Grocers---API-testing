# 🌐 Proyecto de Pruebas: Urban Grocers API  

Este repositorio contiene las **pruebas de API** realizadas sobre la plataforma **Urban Grocers**, incluyendo endpoints de kits y servicios de entrega.  

## 📂 Contenido
- ✅ Colecciones de Postman  
- 🧪 Casos de prueba de API (funcionales, valores límite, negativos)  
- 🐞 Reporte de bugs encontrados  
- 📸 Evidencias de ejecución  

## 🛠️ Herramientas utilizadas
- Postman (ejecución de pruebas)  
- Google Sheets (documentación de casos)  
- Jira (registro de bugs)  

## 👩‍💻 Rol desempeñado
- Creación de colecciones y pruebas en Postman  
- Diseño de casos de prueba (positivos, negativos y valores límite)  
- Registro de errores con evidencia
